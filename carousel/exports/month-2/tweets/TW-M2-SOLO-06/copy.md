Fix these 6 things before you send your next sync pitch:

---
### TWEET  ⚠ 281 chars (over 280 limit)

1. Rights chain documentation (confirmed in writing)
2. Lossless WAV files for every pitched track
3. Full stems organised and cloud-accessible
4. ISRC codes assigned to every recording
5. Mood and BPM tags embedded in file metadata
6. Sub-24-hour response system set up and tested
