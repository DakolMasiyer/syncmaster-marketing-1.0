### 4. CTA

**Narration:**

What we audit:

→ Rights chain documentation (master + publishing: confirmed and provable)

→ Sample clearance (zero tolerance — one uncleared sample stops the whole catalogue)

→ File quality (broadcast-spec WAV — not MP3)

→ Stems organisation (available, named, deliverable in under 2 hours)

→ Metadata completeness (ISRC, BPM, key, mood tags — all embedded)

→ Response infrastructure (sub-24-hour response — demonstrated, not theoretical)
