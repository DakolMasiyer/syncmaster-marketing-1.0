Figma Import Instructions

  • 1
    Open Handoff portal or target file in your web browser.
    We recommend using Google Chrome. Right-click any of the cards above to open the compiled HTML specification page.
  • 2
    Run the html.to.design plugin in Figma.
    In Figma, open a new or existing design file, search for the plugin html.to.design, and open it.
  • 3
    Input the local URL or drag the HTML file.
    Copy the browser's URL (e.g. file:///C:/Users/.../figma-handoff/ui_kits_figma_ready.html) and paste it into the plugin's URL field, or upload the HTML file directly. Set the viewport width matching the target metadata (e.g., 1440px for UI Kits and Design System, 1080px for carousels).
  • 4
    Import and customize!
    Click Import. The plugin translates all HTML elements, fonts, colors, and layout nodes into fully-editable, pixel-perfect Figma layers automatically.
