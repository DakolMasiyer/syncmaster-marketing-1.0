# SyncMaster — 5 NotebookLM Notebooks Setup Guide

Create each notebook separately at notebooklm.google.com.
Every notebook follows the same 4-step pattern:
1. Create notebook with the name shown
2. Upload the listed files (each as a separate source)
3. Add the standing instruction as a Note → Convert to source
4. Use the web search prompt to pull external sources
