# Project Instructions: SyncMaster Marketing 1.0

This workspace is dedicated to marketing assets, social content generation, and sales enablement for the SyncMaster platform.

## Project Context
- **Focus:** High-conversion marketing copy, social media carousels, and automated sales playbooks.
- **Key Tool:** The \/carousel\ directory contains the logic for generating social media assets.
- **Brand:** SyncMaster (Purple \#5252E0\, Lime \#C9E834\, Dark \#0A0A20\, DM Sans).

## Workflow Guidelines
- **Asset Generation:** When creating carousels or social graphics, always verify layout against \web-design-guidelines\ to ensure mobile readability.
- **Copywriting:** Use the \product-marketing.md\ in \.agents/\ as the primary source of truth for positioning and ICP (Ideal Customer Profile).
- **Automation:** For batch playbook generation, utilize \dispatching-parallel-agents\ to process multiple tracks (e.g., Composer vs. Producer) simultaneously.

## Model Coexistence
- This \GEMINI.md\ file contains instructions optimized for Gemini's toolset.
- Shared architectural rules or coding standards found in \CLAUDE.md\ (if present) should be respected, but Gemini-specific tool triggers (like \copywriting\ or \ckmslides\) take precedence for marketing tasks.
